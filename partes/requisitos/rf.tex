
\textbf{Requisitos Funcionais}:

Os requisitos foram priorizados segundo o método MoSCoW, alinhado à abordagem MVP (Produto Mínimo Viável) adotada no projeto: \textbf{[M]} \textit{Must have} — funcionalidades essenciais implementadas no MVP; \textbf{[S]} \textit{Should have} — importantes, mas não críticas para a primeira versão; \textbf{[C]} \textit{Could have} — desejáveis para versões futuras; \textbf{[W]} \textit{Won't have} — fora do escopo do MVP.
