\subsection{Regras de Negócio}

Além dos requisitos funcionais, foram identificadas regras de negócio que impõem restrições ou condições sobre as funcionalidades do sistema:

\begin{itemize}
\item RN001 – Não deve ser permitido acesso a áreas restritas sem autenticação prévia.
\item RN002 – Não deve ser permitido acesso simultâneo com a mesma conta.
\item RN003 – Certificados não podem ser excluídos definitivamente, devendo ser arquivados.
\item RN004 – A emissão de certificados deve ser restrita ao horário configurado para cada perfil de colaborador.
\item RN005 – Certificados só podem ser emitidos para participantes previamente vinculados ao curso ou evento.
\end{itemize}
